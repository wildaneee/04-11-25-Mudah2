# 04-11-25-Mudah2

Penjelasan Program:
Program menggunakan Scanner untuk membaca input dari pengguna.

Operator % digunakan untuk mencari sisa bagi dari bilangan dengan 2.

Jika bilangan % 2 == 0, maka bilangan tersebut genap.

Jika tidak, maka bilangan tersebut ganjil.
